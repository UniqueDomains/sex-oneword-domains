## 🧱 Field summary

- `domain` — Fully qualified domain name.
- `status` — Current acquisition state for the domain in the public extract.
- `purchase_price` — Visible purchase price when available.
- `renewal_price` — Visible renewal price when available.
- `attractiveness` — Composite naming score used as a decision-support signal.
- `demand` — Relative buyer-pressure score when available.
- `length` — Character count without the TLD.
- `registrar` — Registrar name when known.
- `created_at` — Creation timestamp when known.
- `expires_at` — Expiry timestamp when known.

## ⚠️ Methodology and caveats

This set is exclusively made up of .sex domain names, spanning generic terms, personal names, acronyms, and culturally specific words such as Liam.sex, WiFi.sex, homes.sex, LGBTQ.sex, and emoji.sex. The median ask is 158.39, which places many names in a relatively accessible price band, but quality varies sharply. For founders, the main question is whether the word is clear, memorable, and usable without legal friction. For investors, the key test is whether the term has obvious commercial intent, broad keyword recognition, or resale relevance within this exact extension. Generic words usually offer cleaner positioning than branded or trademarked terms.

- Prioritize generic words over branded or trademarked terms
- Check if the word reads clearly and naturally with .sex
- Use median ask 158.39 as a rough pricing reference
- Short, exact-match terms usually carry broader appeal
